Spotify Music Analysis

Описание проекта
Анализ музыкальных данных Spotify для изучения паттернов популярности, аудио-признаков и жанровых особенностей.

Структура проекта
- `Project_Spotify.ipynb` - основной анализ в Jupyter Notebook
- `spotify_songs.csv` - исходные данные
- `requirements.txt` - зависимости Python

Технологии
- Python 3.9+
- Pandas, NumPy
- Matplotlib, Seaborn
- Yandex DataLens
